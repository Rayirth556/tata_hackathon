## Temperature Features
> ⚠️ Temperature columns (`Tavg_mean`, `Tmax_max`, `Tmin_min`, `Tavg_100`) are **not available** in the HUST dataset.
> These are set to `NaN` for all HUST cells.
> Krishiv must use the **`_no_temp`** feature matrix variants when training cross-dataset models.

## Internal Resistance (IR) Note
> ✅ HUST IR is now computed via the **CV→discharge voltage step method**:
> `IR = (V_cv_last − V_dis_first) / (I_dis_first − I_cv_last)`
> The last row of the CV charge phase (current tapering to ~55 mA) serves as the OCV proxy.
> All 77 HUST cells have valid `IR_cycle2`, `IR_cycle100`, and `IR_diff` values.
>
> ⚠️ **Absolute IR values are NOT cross-dataset compatible** — Severson cells are LFP 18650s (~0.013–0.020 Ω)
> while HUST cells have higher base IR (~0.033–0.057 Ω) due to different cell chemistry/form factor.
> Therefore `IR_cycle2` and `IR_cycle100` should NOT be used as joint features.
>
> ✅ `IR_diff` (change from cycle 2 to cycle 100) **does overlap** (77.5%) and is safe for cross-dataset use
> as it captures the *rate of IR change* rather than an absolute offset.

## Capacity Fade Slope
- Severson cells with negative `fade_slope`: **63.4%**
- HUST cells with negative `fade_slope`: **66.2%**
> Over cycles 2–100, HUST cells are largely stable (early-life plateau), so some cells show near-zero or positive slopes — physically consistent.

## Recommended Cross-Dataset Feature Set
Based on availability and range overlap, the following features are recommended for joint Severson+HUST training:

- ✅ `QD_100`
- ✅ `IR_diff` — rate of IR change is cross-dataset compatible (77.5% range overlap)
- ✅ `dVdQ_var_10`
- ✅ `dVdQ_var_100`
- ✅ `dVdQ_var_diff`
- ✅ `I_var_diff`
- ✅ `chargetime_s_mean_2to6`
- ✅ `fade_slope`
- ✅ `fade_intercept`

Features to **exclude** from cross-dataset models:
- ❌ `IR_cycle2` — absolute values not compatible (different cell chemistry offset)
- ❌ `IR_cycle100` — same reason as above
- ❌ `I_var_10`, `I_var_100` — non-overlapping ranges (Severson: constant 4C, HUST: multi-stage)
- ❌ `Tavg_mean`, `Tmax_max`, `Tmin_min`, `Tavg_100` — not available in HUST

Features available for **within-dataset models only**:
- ⚠️ `IR_cycle2`, `IR_cycle100` — valid within each dataset separately
- ⚠️ `I_var_10`, `I_var_100` — valid within each dataset separately
- ⚠️ Temperature features — Severson only
